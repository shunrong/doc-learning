📖 使用说明

  1. 启动服务器： 在终端运行 pnpm server
  2. 连接服务器： 点击"连接服务器"按钮，等待连接成功
  3. 开始编辑： 在文本框中输入内容
  4. 测试协同： 打开多个浏览器标签（或不同浏览器），同时编辑文档
  5. 观察效果： 查看右侧操作历史，观察本地操作（紫色边框）和远程操作（绿色边框）的同步
  6. 测试冲突： 在多个标签中同时插入或删除文本，观察 OT 如何解决冲突
