Demo 4: Delta 数据结构深度解析

      

        Delta 是 Quill 使用的文档数据格式，它是一个基于 JSON 的格式，用于描述富文本内容和变更操作。
      


      
Delta 的三种操作类型：

      

        
1. Insert（插入）

        
{ insert: "Hello", attributes: { bold: true } }

        
插入文本 "Hello" 并应用粗体格式

      


      

        
2. Delete（删除）

        
{ delete: 5 }

        
删除 5 个字符

      


      

        
3. Retain（保留）

        
{ retain: 3, attributes: { italic: true } }

        
保留 3 个字符，并应用斜体格式

      


      
完整示例：

      

        
文档内容的 Delta 表示：

        
{
  "ops": [
    { "insert": "富文本编辑器", "attributes": { "bold": true, "size": "large" } },
    { "insert": "\n", "attributes": { "header": 1 } },
    { "insert": "这是一个普通段落。" },
    { "insert": "\n" },
    { "insert": "这是", "attributes": { "italic": true } },
    { "insert": "斜体", "attributes": { "italic": true, "color": "#667eea" } },
    { "insert": "文本。" },
    { "insert": "\n" }
  ]
}
